# curso-prisma

Comandos prisma:

- Instalar:
  npm i -D prisma

- Inicializar no projeto:
  npx prisma init
  npx prisma generate

- Interagir com o Banco de dados:
  npx prisma db pull

- Acessar Prisma Studio
  npx prisma studio

- Comandos de migração:
  npx prisma migrate dev
  npx prisma migrate deploy
